# Generic Courses

Project created as a way to learn aspnetCore Razor and MVC.

## Run Locally

Clone the project

```bash
  git clone https://github.com/CaioCDJ/GenericCourses.git
```

Go to the project directory

```bash
  cd GenericCourses
```

Install dependencies

```bash
  npm install
```

```bash
  dotnet restore
```

Start the server

```bash
  dotnet run --prject ./src/GenericCourses.Web/
```
